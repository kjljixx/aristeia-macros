# Usage
1. Open Word
2. Press Alt+F11 (or Option+F11 on Mac) to open the VBA window
3. Click on "Normal" in the left sidebar
4. Click on Insert->Module
5. Double-click the new module to open the code window
6. Paste the macro 'aristeia.bas' in the code window
7. Ctrl+S to save (Command+S on Mac)
8. Close the VBA wiindow.
9. Paste AI-generated contention with style tags into Word. Make sure the text you just pasted is highlighted
10. Go to View->Macros and search "FormatStyleTags". Press "Run"
11. Your cards should be formatted nicely!
# Extra: If you want buttons to activate the macros
12. In Word, go to File -> Options -> Quick Access Toolbar (On Mac: Word -> Settings -> Ribbon & Toolbar -> Click 'Quick Access Toolbar' at the top).
13. Change 'Choose Commands from:' dropdown to 'Macros'
14. Select 'Normal.aristeia.FormatStyleTags' and press the 'Add >>' button ('>' on Mac)
15. Do the same for 'Normal.aristeia.DeformatStyleTags'
16. If desired, rename/change the icon for both macros
17. Press OK
18. Buttons should show up at the top of word
